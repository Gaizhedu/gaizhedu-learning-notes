# 元组
概念：元组（tuple）是一种无法改变的序列，一旦元组写入数据，将无法被任何形式改变，适合储存一些固定的数据

> 相当于把字刻在石头上，没办法修改石头上的文字

元组主要由**圆括号()**和**逗号,**构成，各个元素之间用逗号隔开

## 元组的不可变性
每个元组内部的元素无法修改，但如果元组内的元素是一个可以被改变的元素（例如列表），则可以对元组内的元素进行修改

例如：
``` Python
a = (1, 2, [3, 4])
print(a[0],a[2][1])
a[2][1] = 5
print(a[0],a[2][1])
# 输出
1 4
1 5
```
